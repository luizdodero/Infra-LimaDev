# 05_migracao_vps3

## Objetivo
Preparar migracao limpa para o VPS 3 como repositorio mestre de desenvolvimento e governanca.

## Pastas

- `inventario`: lista de pastas, repos e dependencias atuais.
- `saneamento`: limpeza de arquivos obsoletos e redundantes.
- `estrutura_alvo`: desenho final de diretorios no VPS 3.
- `sync_backup`: rotina de sincronizacao entre local, VPS 2 e VPS 3.

## Checklist inicial

- [ ] Levantar inventario completo do estado atual.
- [ ] Marcar o que deve migrar, arquivar ou descartar.
- [ ] Definir plano de corte por ondas (por projeto/frente).
- [ ] Definir rotina de backup antes e apos migracao.
