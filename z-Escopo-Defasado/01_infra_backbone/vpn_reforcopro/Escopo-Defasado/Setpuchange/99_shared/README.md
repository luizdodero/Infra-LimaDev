# 99_shared

## Objetivo
Concentrar recursos reutilizaveis por todas as frentes.

## Pastas

- `scripts`: automacoes operacionais reutilizaveis.
- `templates`: modelos de documento, checklist e padroes de configuracao.

## Regra

Nada aqui deve depender de um projeto especifico.
Se houver dependencia de frente/projeto, mover para a pasta de origem.
